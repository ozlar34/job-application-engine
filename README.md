# job-application-engine

> 🚧 Scaffolding in progress — full README lands in Phase 5.

An end-to-end, agent-driven system for crafting job applications: tailoring CVs and
cover letters to a specific job description, gated by defensibility checks, with a
learnings loop that improves the system after every application.

Built as a set of [Claude Code](https://claude.com/claude-code) skills. This repo
showcases the **crafting** half of the pipeline; the **sourcing/scoring** half lives
in [job-match-radar](https://github.com/ozlar34/job-match-radar).

**All data in this repo is synthetic.** The persona (Alex Rivera), companies, and
metrics are fictional examples that demonstrate the system without exposing real
application content.
